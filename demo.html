使い方

  1. ブラウザ上で Linux が起動するのを待ちます。 ~ $ というプロンプトが出たら起動完了です。（ご利用のコンピュータの性能によりますが 10 秒程度）
  2. 次のように打ち込んでください（最後にリターン）。ファイルの中身を確認できます。 
    cat qlobe.rb
  3. 次のように打ち込んでください。qlobe.rb ファイルを Ruby プログラムとして実行します。 45 度回った地球のプログラムが表示されたら成功です。（5 秒程度） 
    ruby qlobe.rb
  4. より詳しくは、README ファイルをご覧ください。 
    show README
